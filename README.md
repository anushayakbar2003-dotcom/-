# Smart Finance Platform

A React-based FinTech app that transforms e-commerce data into financial products with personalized recommendations and portfolio management.

## Live Demo
[View the app here](https://your-vercel-url.vercel.app)

## Features
- User financial profile (risk tolerance, horizon, budget, liquidity)
- Dynamic recommendation engine
- Product listing with 6 filters
- Return projection calculator
- Portfolio with weighted return and risk distribution

```bash
git clone https://github.com/YOUR_USERNAME/smart-finance-platform.git
